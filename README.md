# UB Rukun Barokah - Aplikasi Manajemen Koperasi Syariah

Aplikasi manajemen koperasi syariah berbasis web untuk pengelolaan simpanan, pinjaman (pembiayaan), saham, dan bagi hasil.

## Fitur Utama
- **Multi-role**: Admin, Koordinator, Nasabah
- **Simpanan**: Mudharabah (bagi hasil), Wadi'ah (titipan), Umroh/Haji
- **Pembiayaan**: Pengajuan, angsuran, denda, pelaporan
- **Saham**: Kepemilikan, pemupukan nilai, sertifikat A4
- **Laporan**: Ikhtisar keuangan, ringkasan dana
- **Responsive**: Tampilan mobile (nav bawah) & desktop (sidebar)
